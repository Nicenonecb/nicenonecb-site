# Scramble Text Lab

## 特效名称

Scramble Text Lab / Signal Decrypt Text

## 具体特效

一个命令行风格的文字扰动和解密动画。标题会先显示为块状字符、符号、数字或盲文字符，再按帧逐字揭示成目标文案。界面包含同步进度、命令行提示、字符雨、字符集切换按钮和 entropy 仪表。

## 实现原理

- 用 React state 保存当前字符集、payload、帧数和 runId。
- 每个字符根据当前帧、行号、字符位置和 runId 生成稳定伪随机值，在揭示前显示扰动字符。
- 每个字符有独立的延迟揭示时间，形成从左到右逐步解密的节奏。
- 到达当前 payload 的帧上限后短暂停留，再自动切换到下一组文案。
- CSS Module 负责扫描线、字符雨、终端面板和按钮状态，组件根节点暴露 CSS 变量给卡片预览控制高度和字号。

## 给 AI 的提示词

```text
请实现一个 React 文字扰动特效，风格像暗色命令行/信号解密面板。标题有多行 payload，初始由随机字符组成，然后按帧逐字揭示成目标文字。支持 blocks、symbols、numbers、braille 多种字符集切换，切换时重新播放动画。界面包含同步进度、命令行提示、扫描线、字符雨、执行按钮、seed 和 entropy 仪表。动画要使用稳定伪随机函数，避免每次 render 闪烁；标题区域要固定高度，避免揭示时布局跳动。请用 CSS Module 写样式，并添加简明中文注释说明核心逻辑。
```
