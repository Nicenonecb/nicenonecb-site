# Charging Sparks

参考 `https://x.com/damotime/status/2054584815849021838` 里的 PopcornFX 风格 idle。原视频约 20.6 秒，画面是黑灰雾化背景中一个小型白热能量核不断搓动，按品红、黄绿、蓝青几个相位爆出简单图形粒子，并偶尔出现穿过核心的长斜向光刺。

## 视觉拆解

- 中心核心：白热不规则碎片团块，持续漂移、旋转、呼吸和高频闪烁，不能停在固定点。
- 简单图形：圆片、软圆、胶囊短条、三角碎片、短线火花。
- 长光刺：低频出现的细长斜线，快速闪出、滑动并衰减。
- 色彩节奏：品红为主，穿插黄绿、蓝青和暖橙的短促阶段。
- 背景：低对比黑灰舞台，中心附近有软雾和轻微 grain，突出发光粒子。

## 实现方式

- `charging-sparks.tsx` 使用 Canvas 2D 在 640x360 参考坐标系中实时绘制。
- 参考视频按 0-20 秒重新抓帧后，核心被改成连续能量场，而不是离散关键帧。
- 每帧都从 `getEnergyState(time)` 计算中心漂移、色彩相位、呼吸、旋转和放电强度，再绘制黑灰径向背景、中心 halftone 方格纹理、彩色烟雾、长斜向 beam、胶囊、圆片、短线火花、ribbon、电弧触须和碎片核心。
- 发光感主要来自 `shadowBlur`、`screen` 合成和径向渐变；粒子使用确定性重生，既持续变化又不会随机跳帧。
- `variant="preview"` 会放大主体构图，让 gallery 卡片里仍然能看清核心和胶囊形状。

## 调参入口

- `loopDuration`：对齐参考视频的循环长度。
- `colorTimeline`：20.6 秒循环里的品红、黄绿、蓝青相位。
- `getEnergyState`：控制核心连续漂移、呼吸、旋转和当前颜色。
- `drawEnergyCore`：控制白热碎片核心、内层高光和变形轮廓。
- `drawPlasmaRibbons` / `drawElectricTendrils`：控制环绕光带和类似搓能量球的细电弧。
- `drawBackground`：控制黑灰背景、中心烟雾和 halftone 方格纹理。
- `charging-sparks.module.css`：控制背景雾、暗角、grain 和详情/预览尺寸。

## 验证重点

- `/effects/charging-sparks` 中核心应始终居中，画面不应空白。
- `/effects` 的预览卡片应保持 16:10 框内布局，不溢出或挤压卡片文字。
- 动画应出现品红、黄绿、蓝青三个明显色彩阶段。
- 约每几秒应能看到一根长斜向光刺穿过核心。
- 移动端宽度下，画面仍保持中心构图，文字和预览不重叠。
